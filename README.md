# PrntscGen
A software which tries to randomly generate a Prntsc (Lightshot) image





# Requirements
.NET Framework 4.8

Segoe UI font

DLL files: Microsoft.Web.WebView2.Core.dll, Microsoft.Web.WebView2.WinForms.dll, Microsoft.Web.WebView2.Wpf.dll, WebView2Loader.dll
